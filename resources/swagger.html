URL Shortener Service

This is a collection of APIs for a URL shortener service that does the following:

  1. User registration
  2. User authentication and token generation
  3. Shorten URLs (with user ownership)
  4. Return a list of all URLs shortened for a User
  5. Delete shortened URLs
